FACTORIZATION METHOD

Example:
Find the truth set of the equation x2 - 2x = 3

Solution:

x2 - 2x = 3
x2 - 2x - 3 = 0
x2 - 3x + x -3 = 0
(x2 - 3x) + (x -3 )= 0
x ( x - 3 ) + 1 (x - 3 ) = 0
( x - 3 ) (x + 1 ) = 0
⇒ x = 3 0r x = -1
Hence the solution set is {x : x = -1 0r 3}
